# Etiqueta de Privacidad — Vape 2 Go

**Última actualización: 22 de junio de 2026**

Un resumen rápido y honesto de qué datos recolectamos y para qué. Para el detalle
completo, lee la Política de Privacidad.

**Garantías:**
- ✓ No vendemos tus datos.
- ✓ No entrenamos IA con tus datos personales.
- ✓ Cada usuario solo ve su propia información.

| Dato | Para qué | Con quién se comparte |
|---|---|---|
| Nombre y correo | Crear y gestionar tu cuenta | Supabase |
| Teléfono | Coordinar entregas y soporte | Supabase, PedidosYa |
| Dirección de entrega | Entregar tus pedidos | PedidosYa |
| Historial de órdenes | Tu cuenta, lealtad y soporte | Supabase |
| Navegación (páginas, producto) | Medir conversión y mejorar la tienda | PostHog |
| Origen de visita (UTM/referido) | Atribuir cupones y campañas | Supabase, PostHog |
| IP aproximada / navegador | Seguridad y anti-fraude | Cloudflare, Sentry |

**IA:** usamos Anthropic (Claude) solo para redactar descripciones de productos
del catálogo — nunca con tus datos personales. Las decisiones que te afectan
tienen revisión humana.

**Tus derechos:** acceso, rectificación y eliminación escribiendo a
**privacidad@vapes.do** (Ley 172-13, RD).
